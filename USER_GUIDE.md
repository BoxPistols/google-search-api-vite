## 基本的な使い方

### 1️⃣ 検索キーワードを入力

![検索フォーム](https://via.placeholder.com/600x100?text=Search+Form)

```
例: React Next.js TypeScript
```

- **複数キーワード**: スペース区切りで入力
- **自動計算**: 入力すると消費クエリ数が自動表示されます

### 2️⃣ 消費クエリを確認

画面に表示される情報：

```
┌─────────────────────────────────────┐
│ キーワード数: 3個 → 消費クエリ: 6   │
│                                     │
│ [消費予定: 6クエリ] [残り: 44クエリ]│
└─────────────────────────────────────┘
```

**計算方法**:
```
消費クエリ = 2ページ × キーワード数
```

| 例 | キーワード数 | 消費クエリ |
|---|------------|----------|
| `React` | 1 | 2 |
| `React Next.js` | 2 | 4 |
| `React Next.js TypeScript` | 3 | 6 |

### 3️⃣ 検索を実行

**ボタンの状態**:

- ✅ **緑色**: クォータ十分 → `検索 (-6)`
- ❌ **赤色**: クォータ不足 → `クォータ不足`（押せません）

### 4️⃣ 結果を確認

検索結果には以下が表示されます：

| 項目 | 説明 |
|-----|------|
| 順位 | 1-20位 |
| タイトル | ページのタイトル |
| URL | ページのアドレス |
| 説明文 | 検索結果の説明 |

**追加機能**:
- 📊 **ドメイン分析**: どのドメインが多く表示されているか
- 📋 **検索履歴**: 過去の検索結果を再表示
- 📈 **統計情報**: 総検索回数、総クエリ消費数

## ログインとクォータ

### クォータとは？

**クォータ** = 1日に使える検索回数の制限です。

### 3つのユーザータイプ

| タイプ | アイコン | クォータ | ログイン | 説明 |
|-------|---------|---------|---------|------|
| ゲスト | 🔓 | 10クエリ/日 | 不要 | お試し利用 |
| 一般ユーザー | 🔐 | 50クエリ/日 | 必要 | 通常利用 |
| オーナー | 👑 | 500クエリ/日 | 必要 | 管理者 |

### ログイン方法

#### 🔐 Googleでログイン

1. 右上の「**Googleでログイン**」ボタンをクリック
2. Googleアカウントを選択
3. 権限を許可

**ログイン後の表示**:
```
┌────────────────────────┐
│  👤 あなたの名前        │
│  一般ユーザー           │
│  [50クエリ/日]         │
└────────────────────────┘
```

#### 👑 オーナーアカウント

設定されたメールアドレスでログインすると自動的にオーナーになります：

```
┌────────────────────────┐
│  👤 あなたの名前        │
│  👑 オーナー            │
│  [500クエリ/日]        │
└────────────────────────┘
```

### クォータのリセット

- **リセット時刻**: 毎日0:00（あなたの地域時間）
- **残り時間**: 画面に表示されます

例: `⏰ リセットまで: 5時間30分`

### 警告システム

#### 🟡 警告（黄色）
```
残りクォータが少なくなっています
この検索を実行すると残り2クエリになります
```

#### 🔴 エラー（赤色）
```
❌ クォータが不足しています
検索ボタンが無効化されます
```

## よくある質問

### Q1. 無料で使えますか？

**A.** はい、無料で使えます。

- ゲストユーザー: 10クエリ/日まで完全無料
- ログインユーザー: 50クエリ/日まで完全無料

### Q2. アカウント登録は必要ですか？

**A.** いいえ、不要です。

- アカウントなしで10クエリ/日使えます
- より多く使いたい場合のみ、Googleでログインしてください

### Q3. クォータはいつリセットされますか？

**A.** 毎日0:00（あなたの地域時間）にリセットされます。

### Q4. クォータを増やせますか？

**A.** はい、以下の方法で増やせます：

1. **Googleでログイン**: 10 → 50クエリ/日
2. **オーナーになる**: 管理者に連絡（500クエリ/日）

### Q5. 検索結果が出ません

**A.** 以下を確認してください：

1. ✅ インターネット接続は正常か
2. ✅ キーワードを入力したか
3. ✅ クォータは残っているか（画面で確認）
4. ❌ エラーメッセージが表示されていないか

### Q6. 過去の検索結果を見られますか？

**A.** はい、検索履歴から確認できます。

1. 📋 検索履歴セクションを開く
2. 見たい検索をクリック
3. 過去の結果が表示されます

### Q7. スマートフォンでも使えますか？

**A.** はい、完全対応しています。

- PC、タブレット、スマートフォンすべてで使えます
- 画面サイズに合わせて自動調整されます

### Q8. ダークモードはありますか？

**A.** はい、あります。

右上のアイコン（☀️/🌙）をクリックして切り替えられます。

## 困ったときは

### ログインできない

1. **ブラウザを確認**
   - Cookie、JavaScriptが有効か
   - プライベートブラウジングモードではないか

2. **別のブラウザで試す**
   - Chrome、Firefox、Safari、Edgeで確認

### 検索結果が0件

1. **キーワードを変更**
   - 別のキーワードで試す
   - 英語/日本語を変えてみる

2. **しばらく待ってから再試行**
   - 一時的なエラーの可能性

### クォータが減らない

1. **ページをリロード**
   - F5キーまたは更新ボタン

2. **ブラウザのキャッシュをクリア**
   - Ctrl+Shift+Delete
   - キャッシュをクリア
